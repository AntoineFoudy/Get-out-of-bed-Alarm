If you wake up with your alarm but still struggle to get out of bed this is the Alarm for you.

Set this up on a server in your bedroom and configure the time.

Then set up a client in a different location in your house with this program: {Code yet to be developed}.

Afterwards get your client's IP address and configure the client_IP_address to match.

Now when the time comes for you to wake up the only way you can shut off the alarm is by getting out of bed and going to your client to turn off the alarm.

CONGRATS !!! You are now out of bed and ready to be productive !
